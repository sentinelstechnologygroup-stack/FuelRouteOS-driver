## Notable Finding: TodayShift.jsx Inline Maps URL

`src/pages/TodayShift.jsx` line 96 hardcodes `maps.google.com` directly instead of using `mapsHelper.buildMapsUrl()`. This bypasses the user's default maps app preference (Apple Maps / Waze / Ask).

**Risk:** Low. Works functionally. But ignores the `mapsApp` setting from `AppContext`.
**Recommended fix:** Replace with `mapsHelper.buildMapsUrl(address, mapsApp)` call.
**Priority:** Phase 10 (Maps/Address Helpers) — not a Phase 3 blocker.
