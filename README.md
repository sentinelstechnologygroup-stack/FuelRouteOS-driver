# FuelRouteOS Driver

FuelRouteOS Driver is the mobile driver workflow application for FuelRouteOS by Sentinels FleetOps.

## Status

Base44 export sanitized and prepared for controlled local development.

## Product Scope

- Driver login / shift start
- Home command screen
- Today route / stop workflow
- Universal Scan
- BOL / Pickup Confirmation
- Navigation / Arrival / Geofence workflow
- Site / Tank Verification
- Delivery Entry
- POD / Delivery Confirmation
- Sync Queue
- Dispatch Chat scaffolding
- Vehicle issue reporting
- Pre-trip / post-trip inspection
- Client branding layer
- Legal / Trust pages

## Development Rules

- Do not remove working screens during cleanup.
- Do not delete forms, functions, or features without approval.
- Do not reintroduce Base44 dependencies.
- Do not copy FleetPanda/Panda Box UI.
- Screenshots are workflow reference only, not visual design targets.
- FuelRouteOS must remain an original product.
